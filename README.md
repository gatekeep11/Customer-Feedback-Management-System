Feedback Management System
This Python application offers a simple GUI to manage customer details and their feedback using Tkinter and SQLite. Users can add, view, and delete customers and feedback entries with real-time updates. Designed for ease-of-use, it helps streamline customer feedback collection and analysis.

Features
Add, view, and delete customer records

Submit and manage feedback with categories and ratings

Persistent data storage with SQLite

User-friendly graphical interface

Usage
Run the Python script.

Enter customer details and click "Add Customer."

Select customers to add feedback or delete entries.

View feedback records in the table below.

Requirements
Python 3.x

Tkinter library (usually included with Python)

SQLite3 (included with Python)
